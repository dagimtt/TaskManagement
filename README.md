# TaskManagement
Project Overview
A complete full-stack task management application with role-based access control, real-time collaboration, and comprehensive task tracking.

Tech Stack
Frontend: React + Vite + Tailwind CSS

Backend: ASP.NET Core Web API

Database: SQL Server + Entity Framework Core

Authentication: JWT Bearer Tokens

Key Features
Authentication & Authorization
JWT-based authentication

4 User Roles: Admin, Director, Division, User

Role-based permissions (14 distinct permissions)


Task Management
Create, read, update, delete tasks

Multiple user assignment (many-to-many relationship)

Task status tracking (Pending, In Progress, Completed)

Priority levels (High, Medium, Low)

Due dates with overdue indicators

User Management
User registration and profile management

Active/inactive user status

Role assignment

User statistics and task distribution


Dashboard & Analytics
Task statistics (completion rate, priority distribution)

User performance metrics

Filtering and search capabilities

Responsive design for all devices
